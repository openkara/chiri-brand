## Design System

**Palette**

| Token | Value | Use |
|---|---|---|
| `--obsidian` | `#0a0a0a` | Primary background |
| `--obsidian-mid` | `#111111` | Card / surface background |
| `--obsidian-lift` | `#181818` | Elevated surface |
| `--amber` | `#C17E35` | Primary accent, headings emphasis |
| `--amber-dim` | `rgba(193,126,53,0.5)` | Borders, dividers |
| `--amber-faint` | `rgba(193,126,53,0.15)` | Subtle fills |
| `--cream` | `#F0EAD6` | Primary text |
| `--cream-dim` | `rgba(240,234,214,0.65)` | Secondary text |
| `--cream-muted` | `rgba(240,234,214,0.38)` | Tertiary / labels |
| `--border` | `rgba(240,234,214,0.08)` | Dividers |

**Typography**

| Role | Font | Weight |
|---|---|---|
| Display / Headlines | Playfair Display | 400, 700, 900 |
| Body / UI / Labels | DM Mono | 300, 400, 500 |

**Voice:** Sparse. Cinematic. One idea per screen. Uppercase spaced labels. Italic amber emphasis.

## Tools

### Business Model Working Session (`tools/business-model.html`)
Interactive single-page tool for modeling Chiri's growth with the founder. Covers:
- **Part 1** — Pricing side-by-side (Deck vs. Replacement-Value) + tool stack calculator
- **Part 2** — Bottoms-up capacity planning (engineer headcount → engagement ceiling)
- **Part 3** — Top-down pipeline & growth ramp (94 named accounts, 3 tiers, Y1/Y2)
- **Part 4** — GTM partnership comp & full-time conversion trigger

All inputs are live — every change ripples through the full model in real time. Share button encodes configuration in the URL.
