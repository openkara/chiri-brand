## SLIDE 6: WHAT EVERY COMPETITOR GETS WRONG (New Slide - Sharp Differentiation)
**Headline:** Every existing solution solves one layer. Chiri solves the operating system.

**The core differentiation argument:**
Today's enterprise AI market is populated by single-ecosystem tools. Each one solves a narrow problem inside its own walled garden. None of them can see — or govern — the whole enterprise.

| Platform | What It Does | What It Cannot Do |
|---|---|---|
| Microsoft Copilot Studio | AI agents inside Microsoft 365 / Azure | Requires custom connectors for SAP, Workday, Salesforce; weakest outside Microsoft ecosystem [Sana Labs, 2026] |
| Salesforce Agentforce | CRM-native AI agents on Salesforce data | Cross-system orchestration (ERP, ITSM, HR) requires MuleSoft or custom API work [Sana Labs, 2026] |
| UiPath AI Agents | RPA + LLM for existing automation footprints | Strongest where RPA already exists; limited for greenfield enterprise transformation [Sana Labs, 2026] |

**The Chiri Moat:**
No existing platform (1) sits above all tools simultaneously, (2) captures and replicates the tacit operational knowledge of human workers (Ontology), and (3) deploys governed digital workers inside the tools employees already use — without requiring a new UI.
